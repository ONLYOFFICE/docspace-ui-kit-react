# DocSpace UI Kit

A shared component library for ONLYOFFICE DocSpace that provides:

- reusable React components, hooks, and utilities;
- MobX stores for state management;
- API client and type definitions.

Used across all DocSpace applications: Client, Login, DocEditor, Management, and SDK.

## Installation

```bash
npm install @docspace/ui-kit
```

### Requirements

- Node.js >= 24
- React >= 19

### Quick start

```tsx
import { Providers } from "@docspace/ui-kit/providers";

<Providers url="https://your-docspace.com/api" apiKey="your-api-key" locale="en">
  <App />
</Providers>
```

### Run Storybook locally

```bash
git clone https://github.com/ONLYOFFICE/DocSpace-ui-kit-react.git
cd DocSpace-ui-kit-react
pnpm install
pnpm storybook
```
