# PeopleSelector

PeopleSelector is a searchable, paginated selector for choosing users and groups from the DocSpace system.

### Features

- **Live API mode** ŌĆö Fetches members, groups, and guests from the DocSpace Search API with infinite scroll
- **Tabs** ŌĆö Toggle Members, Groups, and Guests tabs via `withGroups` and `withGuests`
- **Single / multi-select** ŌĆö Controlled by `isMultiSelect`
- **Room scope** ŌĆö Pass `roomId` to filter users with access to a specific room
- **Access rights** ŌĆö Optional access-right dropdown via `withAccessRights`
- **Current user** ŌĆö Highlights the current user with a "(Me)" label via `currentUserId`
- **Exclusions** ŌĆö Hide or disable specific users via `excludeItems` / `disableInvitedUsers`
- **Header / aside / cancel** ŌĆö Fully composable via `withHeader`, `useAside`, `withCancelButton`

### Default

A basic PeopleSelector with default settings.

<Story of={PeopleStories.Default} />

```tsx
import PeopleSelector from "@docspace/ui-kit/selectors/People";

<PeopleSelector
  withHeader
  headerProps={{ headerLabel: "Select Member", onCloseClick: () => setOpen(false) }}
  onSubmit={(items) => console.log(items[0])}
/>
```
