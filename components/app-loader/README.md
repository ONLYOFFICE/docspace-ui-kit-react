# AppLoader

A full-screen loading indicator displayed while the application is initializing. Uses the `Loader` component with the "rombs" animation type.

## Usage

```tsx
import AppLoader from "@docspace/ui-kit/components/app-loader";

<AppLoader />
```

## Features

- **Full-screen overlay**: Centers the loader in the viewport
- **Rombs animation**: Uses the animated rombs loader style
- **Zero configuration**: No props required — renders a consistent loading state
