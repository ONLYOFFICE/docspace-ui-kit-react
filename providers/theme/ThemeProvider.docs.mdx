# ThemeProvider

ThemeProvider resolves the active theme (light/dark/system) and provides it to child components via styled-components context.

## Features

- **Theme Resolution** — Resolves theme based on `initialTheme`, `systemTheme`, and system preference
- **API Integration** — Fetches color theme from API if not provided
- **System Monitoring** — Monitors system theme preference changes
- **RTL Support** — Supports RTL via the `locale` prop

<Story of={ThemeProviderStories.LightTheme} />

## Usage

```tsx
import { ThemeProvider } from "@docspace/ui-kit/providers/theme";
import { ThemeKeys } from "@docspace/ui-kit/enums";

<ThemeProvider initialTheme={ThemeKeys.BaseStr}>
  <App />
</ThemeProvider>
```
