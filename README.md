# @docspace/ui-kit

UI component library for DocSpace.

## Installation

```bash
pnpm add @docspace/ui-kit
```

## Development

### Storybook

```bash
pnpm storybook
```

### Tests

```bash
pnpm test
```

### Build

```bash
pnpm build
```
