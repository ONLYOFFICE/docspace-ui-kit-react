# @docspace/ui-kit

UI component library for DocSpace.

## Installation

```bash
pnpm add @docspace/ui-kit
```

## Usage

```tsx
import { Text } from "@docspace/ui-kit";

<Text fontSize="14px" isBold>
  Hello World
</Text>
```

## Development

### Storybook

```bash
pnpm storybook
```

### Tests

```bash
pnpm test
```

### Build

```bash
pnpm build
```
